# Advent of Code 2025

Repository containing solutions for Advent of Code 2025 puzzles.

## Structure

```
Advent-of-code-2025/
├── day1/
│   ├── solution.js    # JavaScript solution for day 1
│   └── input.txt      # Puzzle input for day 1
├── day2/              # Will be created when needed
└── README.md
```

## Usage

Each day has its own directory containing:
- `solution.js` - The JavaScript solution script
- `input.txt` - The puzzle input data

To run a solution:
```bash
cd day1
node solution.js
```

## Notes

- Solutions are written in pure JavaScript (no external libraries)
- Each puzzle will be added in its own directory as they become available
